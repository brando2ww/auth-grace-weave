# WiseAuto®

Sistema de gestão completo para revendas de automóveis.

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase (backend)

## Desenvolvimento

```sh
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

## Funcionalidades

- Dashboard com métricas de vendas e estoque
- Gestão de contatos e leads
- Chat integrado (WhatsApp, Instagram, Messenger)
- Controle de estoque de veículos
- Campanhas e anúncios
- Sistema de departamentos e atendentes
- Agendamentos e avaliações
- Integrações com marketplaces
